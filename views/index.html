API: URL Shortener Microservice

User Story:

  1. User/You can POST a URL to [base_url]/api/shorturl/new and will receive a shortened URL in the JSON response.
    Example : {"original_url":"www.google.com","short_url":1}
  2. If you pass an invalid URL that doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"}
  3. When you visit the shortened URL, it will redirect to the original link.

Short URL Creation

example: POST [base_url]/api/shorturl/new - https://www.google.com

Example Usage:

[this_project_url]/api/shorturl/1

Will Redirect to:

https://facebook.com/
